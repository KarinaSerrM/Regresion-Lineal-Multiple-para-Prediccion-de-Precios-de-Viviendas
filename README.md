# Regresion-Lineal-Multiple-para-Prediccion-de-Precios-de-Viviendas
Este proyecto aplica regresión lineal múltiple sobre el conjunto de datos kc_house.csv para predecir precios de viviendas. Incluye preparación de datos, construcción del modelo mediante operaciones matriciales, evaluación de desempeño con métricas estadísticas y visualización de resultados.
